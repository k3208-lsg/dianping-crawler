# 🦊 评论采集小助手

一个基于 ADB 的大众点评评论采集工具，通过可爱的 UI 界面轻松抓取用户评论数据。

## ✨ 功能特性

- 🎯 可视化操作界面，简单易用
- 📱 支持 Android 手机（需开启 USB 调试）
- 💾 每 100 条自动分文件保存
- 📊 实时显示采集进度
- 🔔 操作提示弹窗引导

## 📁 项目结构

```
dazong/
├── crawler_ui.py           # UI 界面程序
├── crawlers/

├── data/
│   └── dianping/           # 大众点评数据目录
│       ├── comments_part1_*.csv
│       ├── comments_part2_*.csv
│       └── ...

```

## 🚀 快速开始

### 第一步：下载 ADB 工具

1. 访问 Google 官方下载页面：
   ```
   https://developer.android.com/tools/releases/platform-tools
   ```

2. 下载 Windows 版本：
   ```
   https://dl.google.com/android/repository/platform-tools-latest-windows.zip
   ```

3. 解压到任意目录，例如：
   ```
   C:\platform-tools\
   ```

4. 验证安装（在解压目录打开命令行）：
   ```bash
   adb version
   ```

### 第二步：配置手机

#### 2.1 开启开发者选项

| 步骤 | 操作 |
|------|------|
| 1 | 打开 **设置** |
| 2 | 找到 **系统管理** 或 **更多设置** |
| 3 | 点击 **关于手机** |
| 4 | 找到 **软件版本号**（或"版本号"）|
| 5 | **连续快速点击 7 次** |
| 6 | 提示"您已处于开发者模式"即成功 |

#### 2.2 开启 USB 调试

| 步骤 | 操作 |
|------|------|
| 1 | 返回 **设置** |
| 2 | 进入 **系统管理** → **开发者选项** |
| 3 | 开启 **USB 调试** 开关 |
| 4 | 可选：开启 **USB 安装** |

#### 2.3 连接电脑

1. 用 USB 数据线连接手机和电脑
2. 手机选择 **文件传输/MTP** 模式
3. 如弹出 **"允许 USB 调试吗？"**，勾选"始终允许"并点击 **确定**

#### 2.4 验证连接

在电脑上运行：
```bash
adb devices
```

预期输出：
```
List of devices attached
XXXXXXXX    device
```

> ⚠️ 如果显示 `unauthorized`，请在手机上确认 USB 调试授权

---

### 第三步：安装 Python 依赖

```bash
# 本项目仅使用 Python 标准库，无需安装额外依赖
# 只需确保 Python 3.8+ 已安装
python --version
```

### 第四步：修改配置（如需要）

编辑 `crawlers/dianping.py`，修改以下配置：

```python
# ADB 路径（如果解压到其他位置）
ADB_PATH = r"C:\Users\你的用户名\Downloads\platform-tools\adb.exe"

# 设备 ID（运行 adb devices 查看）
DEVICE_ID = "你的设备ID"
```

---

### 第五步：运行程序

```bash
python crawler_ui.py
```

## 📖 使用教程

### 1. 启动程序

双击运行 `crawler_ui.py` 或在命令行执行：
```bash
python crawler_ui.py
```

### 2. 检测设备

点击 **「检测设备」** 按钮，确认手机已正确连接。

### 3. 设置采集数量

在「采集数量」输入框中输入目标条数，例如 `1000`。

### 4. 开始采集

1. 点击 **「开始采集」** 按钮
2. 在手机上完成以下操作：
   - 打开大众点评 App
   - 搜索目标商家
   - 进入商家详情页
   - 点击进入评论页面
3. 点击弹窗中的 **「我已完成」** 或等待倒计时结束

### 5. 等待采集

- 保持手机屏幕常亮
- 不要切换到其他 App
- 可随时点击「停止」结束采集

### 6. 查看数据

采集完成后，数据保存在：
```
data/dianping/comments_part1_20260221_180000.csv
data/dianping/comments_part2_20260221_180100.csv
...
```

## 📊 数据格式

每个 CSV 文件包含以下字段：

| 字段 | 说明 |
|------|------|
| content | 评论内容 |
| crawl_time | 采集时间 |

示例：
```csv
content,crawl_time
"服务态度很好，手机货源充足",2026-02-21 18:00:01
"店员很专业，解答很详细",2026-02-21 18:00:02
```

## ⚠️ 注意事项

1. **手机屏幕保持常亮**：采集过程中请勿让手机息屏
2. **不要切换 App**：保持在大众点评评论页面
3. **首次连接**：需要手机确认 USB 调试授权
4. **数据分批保存**：每 100 条自动保存一个文件，防止数据丢失
5. **合法使用**：请遵守相关法律法规，仅用于学习研究

## 🔧 常见问题

### Q: 显示"设备未连接"？

**A:** 检查以下几点：
1. USB 线是否连接好
2. USB 调试是否开启
3. 手机上是否授权了此电脑
4. 运行 `adb devices` 查看设备状态

### Q: 采集数据为 0？

**A:** 请确保：
1. 手机已打开大众点评 App
2. 当前页面是评论/评价页面
3. 评论内容已加载出来

### Q: 程序闪退？

**A:** 可能原因：
1. Python 版本过低（需要 3.8+）
2. ADB 路径配置错误
3. 设备 ID 不正确

### Q: 采集速度慢？

**A:** 正常现象，每次采集需要：
1. 等待页面加载
2. 解析 UI 结构
3. 滑动页面

预计 1-2 条/秒的速度。
